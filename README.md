# Wild Skies

Landing page for Wild Skies, a Hytale mod.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
```

Static files output to `out/`

## Docker

```bash
# Build and run
docker compose up --build -d

# Stop
docker compose down
```

Runs on [http://localhost:3000](http://localhost:3000)
